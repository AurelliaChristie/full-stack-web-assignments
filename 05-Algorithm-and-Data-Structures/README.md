## Spec
Kamu adalah seorang mahasiswa IT yang baru memulai perjalanan, kemudian kamu juga sudah belajar tentang adanya `pseudocode`. Kemudian dosenmu memmberikan sebuah tugas untuk membuat sebuah `pseudocode` tentang cara kerja kamu sebuah sistem antrian.

## Key Learning Points
- peserta dapat menganalisa sebuah permasalahan yang ada dilapangan
- menerapkan pola pikir komputasi
- paham dan dapat membuat sebuah `pseudocode`
